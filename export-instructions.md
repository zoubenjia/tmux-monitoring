# 数据库导出操作指南

## 🚀 快速导出步骤

### 1. 设置数据库用户名
```bash
export DB_USER="your_actual_username"
```
> 请将 `your_actual_username` 替换为你的实际数据库用户名

### 2. 执行导出
```bash
./auto-export.sh
```

### 3. 输入密码
- 脚本会提示输入数据库密码
- 输入时密码不会显示在屏幕上（安全特性）
- 输入完成后按回车键

## 📊 导出内容

脚本将自动导出：
- **config 数据库**：18个表 + 数据库结构
- **CRM 数据库**：18个表 + 数据库结构
- **总计**：38个文件

## 📁 导出位置

数据将导出到：
```
~/personal-data/database-exports/auto_export_YYYYMMDD_HHMMSS/
```

## 🔍 导出验证

导出完成后会显示：
- ✅ 成功导出的表数量
- ❌ 失败的表数量  
- 📁 导出目录位置
- 📋 详细的导出报告

## ⚠️ 注意事项

1. **网络连接**：确保能访问 AWS RDS
2. **磁盘空间**：确保有足够空间存储导出文件
3. **权限**：确保数据库用户有读取权限
4. **时间**：导出可能需要几分钟时间

## 🆘 故障排除

**连接失败**：
- 检查用户名和密码
- 确认网络连接正常
- 验证数据库访问权限

**导出失败**：
- 检查磁盘空间
- 确认表名是否正确
- 查看错误日志
